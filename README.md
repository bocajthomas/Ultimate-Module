# S23 Series Ultimate Module
![S23SUM](https://img.shields.io/badge/Ultimate%20Module-h?label=S23%20Series&color=darkgreen) ![Version](https://img.shields.io/badge/Version%203.6-h?color=dodgerblue)

The Ultimate S23 module aims to port features which are exclusive to newer devices, to S23 series devices.

> [!NOTE]
> There is also an ultimate module for S22 series devices. Please see the [S22 branch] of this repo for the Ultimate S22 module.

> [!IMPORTANT] 
> The Ultimate module currently supports these devices:
> * S23 (SM-S911 / dm1q)
> * S23+ (SM-S916 / dm2q)
> * S23 Ultra (SM-S918 / dm3q)
